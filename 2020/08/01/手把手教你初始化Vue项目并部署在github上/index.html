    三.Git原理简介

    git仓库分为本地仓库与远程仓库
    从本地仓库将文件git到远程仓库流程：工作区—-> 暂存区 —-> 仓库区 —-> 远程仓库（如图）

    Workspace：工作区
    Index / Stage：暂存区
    Repository：仓库区（或本地仓库）
    Remote：远程仓库

    安装Vue

    进入Vue官网，点击[起步]，下载开发版本Vue

    安装Vue脚手架
    bash

    来到克隆项目的文件夹上一级

    bash

    会出现很多选项，一个一个看

    1. 已经存在项目文件夹，是否向文件夹内添加配置文件（Y，回车）
    2. 项目名称（默认为文件夹名称，可修改。回车）
    3. 对项目的描述（可写可不写。回车）
    4. 作者（默认github用户，可修改。回车）
    5. 编译规则（默认第一项，运行时编译。回车）
    6. 是否安装vue-router（Y。回车）
    7. 是否使用ESLint对代码工整度进行检查（Y。回车）
    8. 代码检测规范（默认第一项，standard。回车）
    9. 是否进行单元测试（N。回车）
    10. 是否进行e2e端到端测试（N。回车）
    11. 包管理工具（默认第一项，npm。回车）

    开始自动下载依赖，创建项目
